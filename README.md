# FinanceApp — Frontend

Personal finance PWA built with Next.js 16, React 19, and Tailwind CSS v4.

## Variables de entorno

Crea un archivo `.env.local` en la raíz del proyecto con las siguientes variables:

```env
NEXT_PUBLIC_API_URL=https://tu-backend.example.com/api
```

| Variable | Requerida | Descripción |
|---|---|---|
| `NEXT_PUBLIC_API_URL` | Sí | URL base de la API REST del backend |

## Desarrollo

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
npm start
```

## Lint

```bash
npm run lint
```
